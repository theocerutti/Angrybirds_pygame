

	Projet ISN  -- Théo Cerutti -- Hugo Mani -- 2017-2018
	
	Type du jeu: Jeu des catapultes
	
	Vous pouvez lancer le jeu par deux moyens : 
		-> lancer le jeu directement grâce au code source
		-> installer le jeu grâce à l'installateur
		
	Dossiers:
		others: vous trouverez ici toutes les anciennes versions ainsi que tous les tests
		src: abréviation de SouRce Code, c'est ici que vous pourrez lancer votre programme dans sa version finale
		installeur: .exe permettant d'installer le jeu sur l'ordinateur
		
		
	/!\
	/!\ Veillez à installer la font 'Angrybirds' ( elle est ici -> src/lib/font ) dans le dossier Fonts de Windows.
	/!\
		
	Version du programme : 2.5
